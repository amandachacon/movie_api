Amanda's RomCom Movie API

This an API featuring Romantic Comedies. The API enables users to save their favorite RomComs and learn more about the movie, the director, and the subgenre.

Request URL Method Request Data Format Response Data Format
Show/list all movies. /movies GET none JSON object that holds data of all movies
Show details of a spesific movie. /movies/[id] GET none JSON object that holds details of specific movie.
{

}
Show genre details /movies/[genre] GET none JSON object with genre details.
Show director details /movies/[director] GET none JSON object with director details.
